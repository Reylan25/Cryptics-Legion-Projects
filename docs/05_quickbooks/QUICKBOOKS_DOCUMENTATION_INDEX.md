## ⚡ 5-Second Answer Finder

| Question | Answer | Doc |
|----------|--------|-----|
| What are the 3 values I need? | Client ID, Client Secret, Realm ID | Visual Guide |
| How do I get them? | From https://developer.intuit.com | Tutorial |
| Where do I enter them? | Admin → Config → Integration → QB | Visual Guide |
| What happens when I sync? | Expenses are sent to QB as bills | Tutorial |
| My sync shows 0 records | Usually means no GL codes set up | Quick Ref |
| Does this cost money? | QB subscription costs, not integration | Summary |

| Is my data encrypted? | Yes, OAuth2 + encryption | Summary |

## 📋 Quick Fact Sheet

**5 Most Important Things:**

1. **Get 3 credentials from Intuit:** Client ID, Secret, Realm ID
2. **Configure in app:** Admin → Configuration → Accounting Integration
3. **Test connection:** Should show green ✅
4. **Add GL codes:** Admin → Expense Categories
5. **Click sync:** QB Card → Sync button
