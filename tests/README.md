# 🧪 Tests

Automated tests for the Smart Expense Tracker application.

## 📁 Contents

| Test File | Purpose |
|-----------|---------|
| **test_currency_api.py** | Currency API integration tests |
| **test_notification_persistence.py** | Notification persistence tests |
| **test_onboarding_click.py** | Onboarding flow interaction tests |
| **test_onboarding_flow.py** | Onboarding workflow tests |

## 🚀 Running Tests

### Run all tests
```bash
python -m pytest tests/
```

### Run specific test file
```bash
python tests/test_currency_api.py
```

### Run with verbose output
```bash
python -m pytest tests/ -v
```

---

*Tests are organized by feature to aid discovery and maintenance*
